# Elite Portfolio

A modern, responsive portfolio website for creative developers.

## Overview

This project is a single-page portfolio designed to showcase skills, projects, and contact information. It features a clean, dark-themed design with smooth animations and interactive elements.

## Features

-   **Responsive Design:** Looks great on desktop, tablet, and mobile.
-   **Dark Mode Aesthetic:** Sleek, professional appearance.
-   **Interactive Elements:** Custom cursor, hover effects, and scroll animations.
-   **Project Showcase:** Dedicated section to display your work.
-   **Tech Stack Display:** Highlight your skills.

## Technologies Used

-   **HTML5**
-   **Tailwind CSS** (via CDN)
-   **JavaScript** (Vanilla)
-   **Lucide Icons**
-   **Google Fonts** (Inter & Space Grotesk)

## modifications

To customize this portfolio for your own use:

1.  Open `index.html` in a text editor.
2.  Search for specific sections to edit (look for comments like `<!-- EDIT DISINI: ... -->`).
3.  Replace the placeholder text and images with your own content.

## Deployment

This site is ready for deployment on GitHub Pages. simply upload the files to a repository and enable GitHub Pages in the settings.
